Topic Modeling

Posted by Andreas Taylor on Oct 12, 2018

In the last blog we saw how we can look at clustered Tweet data for some qualitative analysis but what if we want to know what the clusters represent? For this we need to use another unsupervised modeling technique such as Latent Dirichlet allocation. The output is groups of words from what the model has identified as topics. The most important parameter to set beforehand is the number of topics and this is pretty subjective. The whole idea is we don’t know much about the corpus we are looking into so clearly we wouldn’t know the exact number of topics. Different people would probably select different cutoffs. For the model there is a balance between worrying about parameters and the utility of the model.
